# Event Discovery & Tracking Tool

This project is a prototype tool that tracks city-based events and stores them in Google Sheets.  
It supports automated updates, deduplication, and expiry-ready logic.

---

## Features
- City-based event tracking
- Configurable data source
- Google Sheets storage
- Deduplication using event hash
- Ready for scheduled automation

---

## Tech Stack
- Python 3.9+
- Google Sheets API
- gspread
- requests
